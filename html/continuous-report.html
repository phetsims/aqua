Continuous Testing Results

Each column is a specific snapshot of SHAs at one point in time (where iteration through all repos showed no changes). Time shown is approximate.

The number shown in each cell is the (rounded) percentage of tests that have passed. A hyphen indicates no tests have been run. The bolder green indicates that all tests have at least one passing run (and no failures), while lighter green may have some sub-tests that haven't been run yet.

Black borders around a cell means it can be clicked to display recorded information about the runs (usually errors). Click anywhere else (besides the dialog and cell) to dismiss. Errors can be copy-pasted out of the dialog.
